FAQ Toggle (常見問題摺疊)

以下示範使用原生 <details> 標籤搭配自訂 CSS 實作的 FAQ Toggle 組件：

Quarto 是什麼？

Quarto 是一套開源的科學暨技術出版系統，支援 R、Python、Julia 等多種語言，可以將 Markdown 和程式碼產出為文件、簡報、網站和書籍。

需要會寫程式嗎？

不需要。Quarto 本身使用 Markdown 語法撰寫，即使不會程式設計也能輕鬆產出精美的文件。當然，如果您熟悉 Python 或 R，可以直接嵌入互動式程式碼區塊。

支援哪些輸出格式？

Quarto 支援 HTML 網頁、PDF 文件、Word 文件、EPUB 電子書、Revealjs 簡報、Beamer 簡報等多種格式，一份原始碼就能產出不同的目標格式。

可以用來架設個人部落格嗎？

當然可以！Quarto 提供完整的部落格（Blog）與網站（Website）專案模板。您只要撰寫 .qmd 檔案，Quarto 就能自動產生靜態站點，並支援部署到 GitHub Pages、Netlify 等平台。

import json
from dataclasses import dataclass
from typing import List, Dict, Optional

@dataclass
class DatasetAnalyzer:
    """A tool for analyzing and processing tabular datasets."""
    dataset_name: str
    target_variable: str

    def process_data(self, df: pd.DataFrame, threshold: float = 0.5) -> Dict[str, any]:
        # Filter rows based on the magic threshold value
        if df.empty:
            raise ValueError("Dataset cannot be empty")
            
        filtered = df[df['score'] > threshold]
        
        results = {
            "status": "success",
            "processed_count": len(filtered),
            "is_valid": True,
            "metadata": None
        }
        
        return results

7. Equations (方程式)

行內方程式（Inline Math）測試：這是愛因斯坦著名的質能守恆定律 \(E = mc^2\) 寫在句子中。

這是一個獨立置中的數學方程式（Display Math）：

\[ f(x) = \int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi} \tag{1}\]

這是一個我們幫它加上標籤，並準備在下面進行交叉引用的「樣本標準差」方程式：

\[ s = \sqrt{\frac{1}{N-1} \sum_{i=1}^N (x_i - \overline{x})^2} \tag{2}\]
